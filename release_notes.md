# 永定生物科技 技術部 KPI 儀表板 v5.3.0 發布說明 (Release Notes)

🎉 **v5.3.0 更新發布！本次升級專注於「四大核心業務板塊」的分類重構，並加入全域的互動式圖表篩選功能，幫助您深度掌握各類業務的資源消耗狀況。**

## 🚀 新功能與強化 (Features & Enhancements)

### 1. 📊 「四大業務板塊」重構與可視化 (Four Main Business Sectors Dashboard)
- **分類架構重組**：將原本扁平化的工作類別，重新定義為四大戰略板塊：**「維修 (Repair)」、「保養 (Maintenance)」、「裝機與專案 (Installation)」、「內部整備 (Refurbishment)」**。
- **指標淨化**：將保養與裝機案件從「維修」領域中分離，使初修率 (FTFR)、返修率 (Recall Rate)、及真實毛利的計算更為精確，不再受到非故障維修案件干擾。
- **全新業務看板**：在儀表板頂端加入「四大業務板塊分析」區塊，清晰顯示各板塊的案量佔比、點數貢獻，以及該分類專屬的營收與毛利數據。

### 2. 🗂️ 互動式板塊全域篩選 (Interactive Global Category Filters)
- **點擊即篩選**：四大業務板塊的卡片全面支援互動點擊！點擊任一板塊卡片，即可一鍵將下方所有數據（SLA 超標率、高頻機型、重點客戶、工程師點數表格等）切換為**僅分析該板塊的業務狀況**。
- **多維複合條件疊加**：支援將「月份/季度」的時間區間過濾與「業務板塊」的分類過濾進行疊加與交叉比對（例如：鎖定分析「上個月的保養專案」或是「本季的設備整新」之效能表現）。
- **過濾狀態提示**：頂部浮動選單隨時反映當前正套用的篩選條件組合，點擊即可一鍵清除回到全景模式。

## 🛠️ 技術更新與其他優化
- 重寫了 `useKpiData.js` 的底層狀態管理 (State Management)，使其穩定支援複合篩選 (Composite Filtering) 機制。
- 修正並加強了元件重新渲染時的效能，確保大型資料集套用多層過濾時依舊順暢。
- 升級版本號至 v5.3.0，更新相依套件庫的相容性。
